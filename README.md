# Application todolist :

NPM install
Npm Start


## API Reference Category

#### Get all Category

```http
  GET /api/category/all
```

#### Post:  create Category

```http
  POST /api/category/create
```


#### PUT: update Category

```http
  PUT: /api/category/update/:taskId
```


#### DELETE: delete Category

```http
  DELETE /api/category/delete/:taskId
```
